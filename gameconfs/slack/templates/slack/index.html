Gameconfs Slack integration

THIS IS AN ALPHA FEATURE!

Gameconfs integration is currently working in a few Slack teams, but it may break or change, and it will most definitely be replaced by a proper Slack app in the future. So if you try this, be warned that you will have to change or uninstall it some time from now.

For this reason, I also request that you do not spread this link to people I don't know.

Setup

Go to the settings for your team and select “Configure Apps”. You may need administrator privileges to do this.

Note that there is a limit of 10 app integrations on free Slack teams, and this counts as one of them.

You should now be on the Installed Apps page. Select “Custom Integrations” at the top.

If you’ve not set up any custom slash commands before, you will see the quite unhelpful message: “You have not created any custom integrations for this team yet! When you connect an account to use with an integration, it will show here.” Go to the Slack App Directory, select “Build Your Own” in the top right, then “Make Your Own (Make a Custom Integration)”, then “Slash Commands”.

If you have set up custom slash commands before, just click on “Add Configuration” instead.

You should now see an interface allowing you to create a new customized Slack command for your team.

Enter “/confs” as your command (or something else to your liking) and click on “Add Slash Command Integration”.

In the following screen, scroll down to Integration Settings and set the following:

  • URL http://www.gameconfs.com/slack/slash
  • Method GET
  • Customize Name Gameconfs
  • Customize Icon if you like, download and use http://www.gameconfs.com/apple-touch-icon–144x144.png
  • Autocomplete help text turn on “Show this command in the autocomplete list”
  • Description Search Gameconfs
  • Usage hint [event name or place]
  • Descriptive Label Gameconfs

Then click “Save Integration”.
