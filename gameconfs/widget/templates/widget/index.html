The Gameconfs Widget

You can add Gameconfs to your own website as a widget. It looks like this:


How to add it to a WordPress blog

Just go to the 'Plugins / Add New' menu in the administration interface of your WordPress installation and search for Gameconfs.

If you want to install it manually, go to the plugin page in the official WordPress plugin directory and follow the instructions there.

Once installed, you can add the widget to your blog and change its parameters in the 'Appearance' / 'Widgets' menu.

How to add it to any website

The widget is easy to integrate. You just need to add two lines to your website's HTML. First, put:

<div id="gameconfs-widget-container" data-nr-months="3" data-place="Europe" data-width="240" data-height="400"></div>

where you want the widget to appear. You may want to change some of those parameters; this is explained below. Then, put:

<script src="http://www.gameconfs.com/widget/v1/script.js" type="text/javascript"></script>

after that div, for instance at the end of your <body> tag.

Parameters

You can customize the widget by changing the parameters of the div. The parameters are:

  • data-nr-months: The number of months from today you want to see in the widget. The value has to be between 1 and 12. Optional. The default is 3.
  • data-place: The continent, country, state or city you want to show events for. Optional. The default is to leave this parameter out, which shows events in the whole world.
  • data-width: The width of the widget in pixels. Optional. The default is 240.
  • data-height: The height of the widget in pixels. Optional. The default is 400.
  • data-no-css: Set this to true if you don't want the widget to use its own CSS. Optional. See below.

If you tell the widget not to use its own CSS, all the styling is entirely up to you. This is an option for advanced users.

Customizing CSS

Here are some tips to help you customize the style of the widget if you've decided to not use the widget's own CSS:

  • The widget is in a div with the ID 'gameconfs-widget-container'.
  • The widget mainly uses h2 and h3 elements, and tables for the event data.
  • You can use the widget's existing CSS for inspiration. You can probably ignore the first section, which was copied from Bootstrap. You can also ignore the heavy use of !important: that's just to make sure the widget's CSS overrides that of the site it's embedded into.

FAQ

Here are the answers to some questions you might have:

How stable is this widget?

I've had this running on my own blog since May 2013, and I've not had any problems with it.

Can your widget damage my site?

I don't see how: it injects some HTML and CSS into the page. The worse case I can imagine is it makes your site look odd. But I make no guarantees.

How does the widget work?

The script uses jQuery to insert its own CSS into your page and to load some HTML from the Gameconfs server (using JSONP). It tries to play nice with your Javascript, jQuery and CSS.

Do you take feature requests?

Within reason, sure. I want this widget to be useful. Just email me and I'll see what I can do.
