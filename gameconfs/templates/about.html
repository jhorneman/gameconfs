About Gameconfs

Gameconfs intends to be a directory of events related to game development, game business, the games industry the medium of games. Never miss another event!

Gameconfs is in early alpha. Things I have planned:

  • Many more events
  • Filtering by location (region, country, state, city)
  • Filtering by keywords (development, b2c, social, bizdev, serious, etc.)
  • Multiple deadlines per event (registration, submission, etc.)
  • Notifications (on new events, before deadlines)
  • Better search
  • Better visual design
  • A logo

Follow Gameconfs news at @Gameconfs.

My name is Jurie Horneman. I occasionally blog at Intelligent Artifice. I regularly tweet as @JurieOnGames. I make games for a living.

Want to help? Do you have questions or feedback? Let me know.

This site was made possible by open source software such as Flask, Python and Bootstrap. It runs on Heroku.
